# Robaczki — p5 Starter

Prosty szkielet projektu używający p5.js.

Pliki dodane:
- `index.html` — strona z przyciskiem Start i kontenerem canvas
- `src/sketch.js` — eksportuje globalnie `Robaczki.start()` i `Robaczki.draw()`
- `src/main.js` — przypina przycisk Start do `Robaczki.start()`
- `package.json` — skrypt `npm start` uruchamia prosty serwer lokalny

Uruchomienie lokalne:

1. Otwórz terminal w katalogu projektu.
2. Uruchom serwer:

```bash
npm start
```

3. W przeglądarce kliknij przycisk "Start" aby uruchomić szkic p5.

Możemy teraz rozbudować `src/sketch.js` — powiedz co chcesz dodać: ruch robaczków, kolizje, sterowanie itp.
